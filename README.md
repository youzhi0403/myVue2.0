3.axios 0.18.0  
4.better-scroll  1.13.2  

# 快速部署

1.clone到本地。

2.安装nodejs环境。（我本地的nodejs版本为v12.13.0）

3.进入项目根目录，运行npm install命令。

4.运行npm run dev命令。

5.通过 http://localhost:8080/ 访问。
